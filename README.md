# POE Part 1 - Java Login System

This project is part of a Portfolio of Evidence (POE) submission.

It is a Java-based application that demonstrates a simple login system using multiple classes. The program includes user authentication, input validation, and basic object-oriented programming principles.

## Classes Included
- LoginClass
- LoginTest
- MainClass

## Purpose
The purpose of this project is to demonstrate understanding of Java programming concepts such as classes, methods, and program structure as required for the POE submission.
